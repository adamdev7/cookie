Configuration des notifications

Remplissez une section à la fois. Les secrets déjà enregistrés restent en place si vous laissez le champ vide.

Boutique

1 · Gmail (confirmations clients)

  1. Activez la validation en 2 étapes sur votre compte Google.
  2. Allez dans Mots de passe des applications.
  3. Créez un mot de passe pour « Courrier », puis collez-le ci-dessous (16 caractères).

2 · Telegram (alertes commandes)

  1. Ouvrez Telegram et parlez à @BotFather → /newbot.
  2. Copiez le token du bot ici.
  3. Envoyez un message à votre bot, puis ouvrez
    https://api.telegram.org/botVOTRE_TOKEN/getUpdates et copiez le chat.id numérique.
  4. (Optionnel) Pour les boutons Accepter/Préparer sur le serveur public, configurez le webhook ci-dessous.

Exemple :
https://api.telegram.org/botTOKEN/setWebhook?url={{ webhook_url }}&secret_token=VOTRE_SECRET

3 · WhatsApp (alertes téléphone)

  1. Utilise CallMeBot (gratuit pour alertes personnelles).
  2. Ajoutez le contact CallMeBot et envoyez le message d'autorisation indiqué sur leur site.
  3. Recevez votre clé API, puis entrez votre numéro (indicatif pays, ex. 15145944092) et la clé.

Tester les envois

Après avoir enregistré, testez chaque canal.

Comment ça marche

  • Gmail envoie la confirmation au client.
  • Telegram vous alerte avec boutons de statut.
  • WhatsApp vous envoie le résumé sur votre téléphone.
